# Инструкция по деплою на Beget

## Подготовка

Билд успешно создан в папке `dist/`. Размер: ~90 MB (включая видео).

## Шаги деплоя на Beget

### 1. Войдите в панель управления Beget

Откройте [https://cp.beget.com](https://cp.beget.com) и авторизуйтесь.

### 2. Создайте или выберите домен

1. Перейдите в раздел **"Домены"**
2. Если нужен новый домен — нажмите **"Добавить домен"**
3. Или используйте существующий домен

### 3. Откройте файловый менеджер

1. Перейдите в раздел **"Файловый менеджер"**
2. Откройте папку вашего домена: `ваш-домен.ru/public_html/`
3. **Удалите** все файлы в этой папке (если есть старые)

### 4. Загрузите файлы

**Вариант A — Через файловый менеджер:**

1. Нажмите **"Загрузить файлы"**
2. Выберите все файлы из папки `dist/`
3. Дождитесь загрузки (может занять время из-за видео)

**Вариант B — Через FTP (рекомендуется для больших файлов):**

1. Подключитесь через FTP-клиент (FileZilla, WinSCP)
2. Данные для подключения найдите в панели Beget → "FTP"
3. Загрузите содержимое папки `dist/` в `public_html/`

### 5. Проверьте .htaccess

Файл `.htaccess` уже добавлен в папку `dist/`. Он обеспечивает:

- ✅ Корректную работу React Router
- ✅ GZIP сжатие
- ✅ Кеширование статических файлов

### 6. Настройте SSL (HTTPS)

1. Перейдите в раздел **"Домены"**
2. Нажмите на шестерёнку рядом с доменом
3. Включите **"Let's Encrypt"** для бесплатного SSL

### 7. Проверьте сайт

Откройте ваш домен в браузере. Всё должно работать!

## Возможные проблемы

### Белый экран

- Проверьте, что `.htaccess` загружен
- Убедитесь, что все файлы в корне `public_html/`

### Картинки/видео не загружаются

- Проверьте права доступа (должны быть 644 для файлов, 755 для папок)

### Ошибка 500

- Проверьте синтаксис `.htaccess`
- Обратитесь в поддержку Beget
